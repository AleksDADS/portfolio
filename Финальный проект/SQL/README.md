## Используемые навыки и инструменты
- Python
- Pandas
- Sqlalchemy
## Описание проекта
- Подключился к базе данных, изучил её и выполнил 5 запросов
## Результат
- Определил сколько книг в базе, рейтинги, обзоры, популярные издательства и авторы
